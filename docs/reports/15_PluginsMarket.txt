模块名称：PluginsMarket.vue（插件市场）
文件路径：src/views/PluginsMarket.vue

功能概述
插件市场页面，展示可安装的插件列表。支持三种插件类型：LobeChat、Gradio、MCP。提供搜索过滤、手动安装（文件/剪贴板/URL）、MCP 插件添加弹窗。在 Web 环境自动过滤掉 MCP 类型插件。

入口与路由
- 路由：/plugins/（PluginsPage 的默认子路由）

核心数据流
1. load() → fetch /json/plugins.{locale}.json → reactive list
2. query + IsTauri → computed filterList（搜索过滤 + Web环境过滤 MCP）
3. install → useInstallPlugin().install() → PluginsStore 安装方法

依赖关系
- composables/install-plugin：统一的插件安装入口
- utils/platform-api：IsTauri 判断平台
- components/InstallPluginBtn：安装按钮组件
- components/PluginTypeBadge：插件类型标签
- components/AddMcpPluginDialog：MCP 插件配置弹窗

关键实现细节
- L58-63：filterList 在非 Tauri 环境过滤掉 MCP 插件（Web 不支持 STDIO）
- L78-89：load() 加载本地化插件列表，支持重试
- L93-102：onFileInput/clipboardImport 调用 useInstallPlugin().install()
- L104-108：addMcpPlugin() 打开 AddMcpPluginDialog

与其他模块的交互
- useInstallPlugin：统一的插件安装入口，自动识别类型
- PluginsStore：最终安装目标
- ATip：MCP 插件使用提示（仅 Tauri 显示）
