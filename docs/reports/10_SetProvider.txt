================================================================================
SetProvider — Provider 设置页面（URL Scheme）
================================================================================
文件路径: src/pages/SetProvider.vue

1. 功能概述
通过 URL scheme 快速设置默认 Provider，支持一键配置和回滚。

2. 入口与路由
- URL: /set-provider?provider=<encoded JSON>
- 通常由 SettingsView 生成的分享链接打开

3. 核心数据流
1. until(userPerfsStore.ready) 等待偏好加载
2. JSON.parse(route.query.provider) 解析 Provider 配置
3. Validator(ProviderSchema) 验证格式
4. 设置 perfs.provider = provider
5. 显示通知，提供"恢复"按钮回退到旧值
6. openLastWorkspace() 导航到最后使用的工作区

4. 依赖关系
- Stores: useUserPerfsStore
- Composables: useOpenLastWorkspace
- 外部: @cfworker/json-schema, @vueuse/core (until)

5. 关键实现细节
- 无模板 (<template><div /></template>)，纯逻辑页面
- 备份旧 provider 到 bak 变量，通过通知 actions 提供恢复
- ProviderSchema 验证确保 type 和 settings 结构正确
- 通知 timeout 6000ms

6. 与其他模块交互
- 修改 perfs.provider 影响全局 Provider 设置
- openLastWorkspace 导航回工作区
