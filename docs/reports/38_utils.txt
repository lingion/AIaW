模块名称：工具函数层（utils/）
文件路径：src/utils/

功能概述
包含 30+ 个工具模块，涵盖数据库定义、类型系统、插件构建、平台适配、文件处理等核心功能。总计约 7000 行代码。

核心模块

1. types.ts（610行）— 类型系统
   - 定义全部 TypeScript 类型：Workspace, Dialog, Message, Assistant, Provider, Plugin, Artifact, CustomProvider 等
   - 插件清单 Schema：LobePluginManifestSchema, GradioPluginManifestSchema, McpPluginManifestSchema
   - JSON Schema 类型：MarketAssistantSchema 等

2. db.ts（139行）— Dexie 数据库
   - 定义 IndexedDB 表：workspaces, dialogs, messages, items, assistants, artifacts, providers, installedPluginsV2, reactives, avatarImages
   - defaultModelSettings：默认模型参数（temperature, maxTokens 等）
   - 可选 Dexie Cloud 同步（DexieDBURL 配置）

3. values.ts（1964行）— 常量和提供商定义
   - ProviderTypes：原生提供商类型列表（OpenAI, Anthropic, Google, DeepSeek 等），每个包含 name, label, constructor, getModelList
   - models：预定义模型列表
   - modelOptions：所有可选模型名
   - dialogOptions：Quasar Dialog 默认样式

4. plugins.ts（931行）— 插件构建工具
   - buildLobePlugin：将 LobeChat 插件清单转为 Plugin 对象
   - buildGradioPlugin：构建 Gradio 插件
   - buildMcpPlugin / dumpMcpPlugin：构建和初始化 MCP 插件
   - 内置插件工厂：timePlugin, calculatorPlugin, whisperPlugin, fluxPlugin, emotionsPlugin, mermaidPlugin 等

5. functions.ts（290行）— 通用工具函数
   - genId：UUID 生成
   - caselessIncludes：大小写不敏感搜索
   - saveArtifactChanges / restoreArtifactChanges
   - blobToBase64, removeUndefinedProps 等

6. config.ts（13行）— 环境配置
   - 从 process.env 读取所有配置：MaxMessageFileSizeMB, DocParseBaseURL, CorsFetchBaseURL, DexieDBURL, LitellmBaseURL, BudgetBaseURL, SearxngBaseURL 等

7. platform-api.ts（90行）— 平台适配
   - IsTauri, IsCapacitor, IsWeb 平台检测
   - fetch：按平台选择实现（tauriFetch / capFetch / window.fetch）
   - clipboardReadText, exportFile：平台特定实现

8. templates.ts（178行）— 提示词模板
   - AssistantDefaultPrompt：默认助手提示词模板（Nunjucks）
   - 支持变量注入：{{prompt}}, {{date}}, {{workspace.vars.*}}

9. middlewares.ts（93行）— AI SDK 中间件
   - FormattingReenabled：o1/o3 系列恢复格式化
   - MarkdownFormatting：GPT-5 系列 Markdown 格式提示
   - AuthropicCors：Anthropic 浏览器直连头

10. template-engine.ts（5行）— Nunjucks 模板引擎单例

插件模块（各 ~100-350行）
- web-search-plugin.ts：SearXNG 搜索插件
- doc-parse-plugin.ts：LlamaParse 文档解析插件
- code-exec-plugin.ts：Python 代码执行插件
- file-ops-plugin.ts：文件操作插件（读写列表执行）
- local-fs-native-plugin.ts：本地文件系统插件
- artifacts-plugin.ts：Artifact 管理
- mcp-client.ts：MCP 客户端
- mcp-sse-transport.ts：MCP SSE 传输
- audio-process.ts, image-process.ts, pdf.ts：媒体处理

辅助模块
- cors-fetch.ts（21行）：CORS 代理请求封装
- sessions.ts（32行）：BroadcastChannel 多标签页会话管理
- local-data.ts（17行）：LocalStorage 用户数据（visited, lastReloadTimestamp）
- update.ts（122行）：版本更新检查
- builtin-plugin-seed.ts（55行）：内置插件种子数据
- codejar.ts（579行）：CodeJar 代码编辑器（第三方库内嵌）
