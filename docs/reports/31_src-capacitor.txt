模块名称：src-capacitor/ — 移动端壳
文件路径：src-capacitor/

功能概述
Capacitor 移动端配置。将 Web 前端打包为 iOS/Android 原生应用。配置应用 ID、名称、Web 目录和 Android Scheme。

入口与路由
- capacitor.config.json：Capacitor 配置文件
- www/：构建输出的 Web 资源目录

核心配置
- appId: "com.nitrorcr.aiaw"
- appName: "AI as Workspace"
- webDir: "www"（Quasar 构建输出）
- server.androidScheme: "https"（Android 使用 https scheme 以支持某些 API）

平台适配（在 src/utils/platform-api.ts 中处理）
- IsCapacitor = Capacitor.isNativePlatform()
- CapacitorPlatform = Capacitor.getPlatform()（'android' | 'ios'）
- fetch：Android 使用 capacitor-stream-fetch，iOS 回退到 window.fetch（未实现）
- clipboardReadText：使用 @capacitor/clipboard
- exportFile：通过 @capacitor/filesystem + capacitor-export-file 写入并导出
- StatusBar/EdgeToEdge：Android 状态栏和边到边显示

依赖关系
- @capacitor/core：核心运行时
- @capacitor/clipboard：剪贴板
- @capacitor/filesystem：文件系统
- @capacitor/status-bar：状态栏
- @capawesome/capacitor-android-edge-to-edge-support：边到边显示
- capacitor-stream-fetch：流式 fetch（仅 Android）
- capacitor-export-file：文件导出

与其他模块的交互
- platform-api.ts：检测 Capacitor 环境并提供平台特定的 API 实现
- set-theme.ts：Capacitor 环境同步 StatusBar 样式
