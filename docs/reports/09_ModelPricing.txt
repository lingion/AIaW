================================================================================
ModelPricing — 模型定价
================================================================================
文件路径: src/pages/ModelPricing.vue

1. 功能概述
展示可用模型的价格表，提供用量计算器（仅中文 locale 显示）。

2. 入口与路由
- URL: /model-pricing
- 仅在 DexieDBURL + LitellmBaseURL 配置存在时注册

3. 核心数据流
1. 未登录 → 重定向 / + 登录
2. fetch LitellmBaseURL/model/info 获取模型列表和价格
3. modelPrices computed 根据 unit (CNY/USD, K/M tokens) 计算显示价格
4. 用量计算器: budget ↔ output 双向计算 (costPerWord 按模型类型乘系数)

4. 依赖关系
- Stores: useUiStateStore
- Components: ModelItem
- Config: LitellmBaseURL, UsdToCnyRate

5. 关键实现细节
- costPerWord: output_cost_per_token × 系数 (gpt-4: 1.4, deepseek: 1.8, 其他: 1)
- modelPrices: 按 modelName 排序
- 去重: Set 过滤重复 model_name
- 仅中文 locale 显示用量计算器

6. 与其他模块交互
- AccountPage 链接到此页面
- 共享 LitellmBaseURL 和用户 apiKey
