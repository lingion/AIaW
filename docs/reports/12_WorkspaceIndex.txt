模块名称：WorkspaceIndex.vue（工作区首页）
文件路径：src/views/WorkspaceIndex.vue

功能概述
工作区首页视图，渲染工作区的 indexContent 字段（Markdown 格式）。使用 Nunjucks 模板引擎解析内容，支持在首页中嵌入 workspace 变量。是用户进入工作区后看到的默认页面。

入口与路由
- 路由：/workspaces/:workspaceId/（WorkspacePage 的默认子路由，path: ''）
- 通过 inject('workspace') 获取当前工作区对象

核心数据流
1. inject('workspace') → syncRef 双向绑定 → workspacesStore.putItem 持久化
2. workspace.indexContent → engine.parseAndRenderSync → computed contentMd
3. contentMd → MdPreview 组件渲染

依赖关系
- composables/sync-ref：双向绑定工作区对象到 store
- composables/md-preview-props：Markdown 预览配置
- composables/set-title：设置页面标题
- stores/workspaces：工作区 CRUD
- utils/template-engine：Nunjucks 模板引擎实例
- md-editor-v3：Markdown 预览组件
- components/ViewCommonHeader：通用页头

关键实现细节
- L10-13：syncRef 将 inject 的 workspace 与 store 双向同步，valueDeep: true 确保深层响应
- L22：computed contentMd 使用 engine.parseAndRenderSync 渲染模板，传入 { workspace } 作为模板上下文
- L25：useMdPreviewProps() 返回响应式的 Markdown 预览主题配置

与其他模块的交互
- WorkspacePage：通过 provide/inject 传入 workspace
- WorkspacesStore：通过 syncRef 持久化修改
- template-engine：渲染带变量的 Markdown 内容
